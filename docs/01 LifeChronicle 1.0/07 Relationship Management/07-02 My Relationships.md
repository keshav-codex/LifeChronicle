07-02 My Relationships
1. Introduction

The My Relationships page displays all active relationships belonging to the authenticated user.

It provides a simple interface for viewing, searching, filtering, and managing personal relationships.

2. Information Displayed

Each relationship card displays:

Profile Photo

Full Name

Relationship Type

Nickname

Additional information is displayed only according to the relative's privacy settings.

3. Available Actions

Users may:

View Relative Profile

Edit Nickname

Put Relationship On Hold

Delete Relationship

Block Relationship

Manage Relationship

4. Search and Filter

Users may search only within their own relationship list.

Supported search criteria:

Full Name

Nickname

Relationship Type

Category Filter

The application shall not support global user searching.

5. Pagination

Relationship lists shall support:

Pagination

Sorting

Responsive layouts

Large relationship lists shall load efficiently.

6. Business Rules

The system shall ensure:

Only accepted relationships appear.

Pending invitations are displayed separately.

Hold relationships are hidden until restored.

Deleted relationships require a new invitation to reconnect.

Blocked relationships cannot exchange invitations until unblocked.

7. Summary

The My Relationships page provides users with a privacy-aware and efficient interface for managing trusted connections while maintaining independent control over relationship status and visibility.
